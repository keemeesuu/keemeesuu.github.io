기본 단축키 모음 1

<br>

### 오브젝트 제어

`A` : 전체를 선택 혹은 해제 할 수 있다.

`B` : 영역 선택이 가능해진다. 드래그 하여 영역을 지정 해 원하는 도형들을 선택한다.


`Shift + A` : 객체 추가

`Shift + D` : 객체 복제

`Delete, X` : 객체 삭제

`S` -> `Shift z` : 오브젝트 폭만 조정하기 `Shift` 누르면 세밀 조정

<br>
<br>

### 뷰포트 제어

`Control Tab` : 에디트 모드

`T` : 툴바 숨기기/보이기

`N` : 사이드바 숨기기/보이기

### 단축키

`Shift + C` : 3D 커서 초기화(0, 0, 0)

`Command + R` : 스컬트 리매쉬의 입자크기를 낮게 설정할수록 메쉬의 디테일을 높힌다.

`F` : 브러쉬 크기 단축키

`Shift F` : 브러쉬 필압의 강도
